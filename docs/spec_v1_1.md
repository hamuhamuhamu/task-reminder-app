# こども記録リマインダー 仕様書 v1.1（タスクモデル）

## 1. 概要
- Android向けの記録管理アプリ
- 通知とアプリ内入力で日次記録を管理
- 各項目について「最終記録からN日」を表示
- 複数タスクを管理可能

## 2. 主機能
- 毎日指定時刻に通知
- 通知欄の `Yes / No` で当日記録
- ホームでタスク一覧を表示
- タスク詳細で `表表示` と `カレンダー` をタブ切替
- 過去日付に遡って入力可能

## 3. 画面遷移
```text
[起動]
  ↓
[ホーム（タスク一覧）]
  ├─ 右下+ → [タスク作成/編集]
  ├─ 設定アイコン → [設定]
  └─ タスクカードタップ → [タスク詳細]
                        ├─ タブ: [表表示]
                        │    ├─ 「あり/なし」クイック入力
                        │    └─ 行タップ → [日次記録編集（対象項目）]
                        └─ タブ: [カレンダー]
                             ├─ 日付タップ → [日次記録編集（日付×項目）]
                             └─ 月切替（前月/次月）
```

## 4. 通知文言
- `（項目名）今日で最終記録からN日です`

## 5. N日ルール
- 基準: 直近 `status = YES` の日付
- `N = 今日 - 最終YES日`
- YESが未登録の場合は `未記録`
